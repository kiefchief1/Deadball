# Deadball Radio Booth

A static responsive web app for solo Deadball play on desktop and mobile browsers.

## What this build includes

- Modern-era solo exhibition game flow
- Radio-style play-by-play commentary
- Magazine-style end-of-game recap
- Local archive of finished games using browser storage
- Campaign hub for journaling pennant chases and season projects
- Sample Southern Circuit clubs plus custom team JSON import
- Optional oddities and injuries toggle

## How to use

### Open locally

Open `index.html` in a browser.

### Host on the web

Upload the whole folder to any static host, such as:

- GitHub Pages
- Netlify
- Cloudflare Pages
- Vercel static hosting

No server code is required.

## Notes

This is a first playable prototype aimed at fast solo play and long-term tracking. It focuses on the modern core game loop and narrative presentation first. The uploaded source material also supports future expansion into:

- Nine Game Pennant campaign automation
- Franchise builder workflows
- season simulation, prospects, and trades
- all-star, legends, and Ring of Honor event packs
- dirty play and other optional chaos rules


Deployment prep added:
- vercel.json for Vercel static hosting
- netlify.toml for Netlify static hosting
- In-app Deploy tab with publishing steps

This environment cannot publish the site live, but the folder is ready to upload to a static host.
